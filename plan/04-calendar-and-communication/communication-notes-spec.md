# Communication Notes Spec

The MVP needs structured notes more than full chat.

## Note Types

- Mentor private note.
- Shared mentor note.
- Mentee note.
- Next action.
- Session summary.

## Visibility

- Private mentor notes are visible only to the mentor and admin if operationally necessary.
- Shared notes are visible to mentor and mentee.
- Mentee notes are visible to the mentee and assigned mentor.

## Context

Notes may belong to:

- Relationship.
- Plan.
- Lesson.
- Session.

## Acceptance Criteria

- Notes are attached to meaningful context.
- Mentors can record decisions and follow-ups.
- Mentees can see shared next actions.

## Global Design Direction

All UI-facing implementation from this spec must follow `plan/07-visual-identity-and-design/`. That folder is the official NossaMentoria visual identity: warm light surfaces, learning-oriented accents, Sora headings, Inter operational text, restrained motion, landing-first public entry, and dashboard-first private usability.
