# Content And Classes

This phase defines how mentors upload materials, organize lessons, and record what happened in class.

## Specs

- `content-library-spec.md`: reusable mentor materials.
- `upload-materials-spec.md`: files and links.
- `lesson-organization-spec.md`: how lessons are structured.
- `class-session-records.md`: session logs and applied content.

## MVP Priority

The platform must support Gabriel uploading slides and `.html` files and attaching them to lessons or plans. Other mentors should get the same capability.

## Acceptance Criteria

- Mentors can organize content before a class.
- Mentors can attach content to a lesson.
- After a class, mentors can record what was actually applied.
