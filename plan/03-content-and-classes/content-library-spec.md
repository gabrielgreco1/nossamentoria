# Content Library Spec

The content library stores reusable learning materials owned by mentors.

## Material Types

- Uploaded `.html` slides or pages.
- PDF files.
- Presentation files.
- Markdown notes.
- External links.
- Code repository links.
- Plain text notes.

## Library Behavior

Mentors can:

- Upload or register materials.
- Add title, description, tags, and category.
- Preview or open materials.
- Attach materials to plans and lessons.
- Reuse the same material across multiple mentees.

## Ownership

Materials belong to the creator. Admin may have operational access. Mentees only see materials explicitly attached to their plans or lessons.

## Acceptance Criteria

- A mentor can find and reuse previous materials.
- Materials are not automatically public to all users.
- Each material records owner, type, title, and source.

## Global Design Direction

All UI-facing implementation from this spec must follow `plan/07-visual-identity-and-design/`. That folder is the official NossaMentoria visual identity: warm light surfaces, learning-oriented accents, Sora headings, Inter operational text, restrained motion, landing-first public entry, and dashboard-first private usability.
