# Success Metrics

The MVP is successful if it makes real mentorship easier to run, not if it generates revenue.

## Activation Metrics

- Invited users can create accounts successfully.
- Mentors can create at least one mentorship plan.
- Mentees can access their assigned plan and materials.
- At least one class can be scheduled and completed.

## Engagement Metrics

- Active mentorship plans have upcoming lessons.
- Mentors update class status after sessions.
- Mentees return to view materials and next actions.
- Plans have visible progress instead of staying static.

## Operational Metrics

- Admin can understand who has access.
- Mentors can understand which mentees need attention.
- Mentees can understand what they should study next.

## Quality Metrics

- No user can access another mentor's private mentee records unless allowed.
- Uploaded materials remain connected to the correct mentor, plan, or class.
- Completed classes preserve notes about what was applied.

## Out Of Scope Metrics

- Revenue.
- Conversion to paid plans.
- Public marketplace growth.
- Advertising performance.

## Global Design Direction

All UI-facing implementation from this spec must follow `plan/07-visual-identity-and-design/`. That folder is the official NossaMentoria visual identity: warm light surfaces, learning-oriented accents, Sora headings, Inter operational text, restrained motion, landing-first public entry, and dashboard-first private usability.
