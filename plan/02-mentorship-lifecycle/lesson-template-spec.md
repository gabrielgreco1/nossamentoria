# Lesson Template Spec

Lesson templates help mentors create plans quickly.

## Template Purpose

Templates are reusable mock lessons that can be copied into a mentorship plan. They are not completed lessons until attached to a plan.

## MVP Template Examples

- Programming fundamentals review.
- Git and GitHub workflow.
- HTML and semantic structure.
- CSS responsive layout.
- JavaScript fundamentals.
- React components and state.
- API consumption.
- Authentication basics.
- Portfolio project review.
- Interview preparation.

## Template Fields

- Title.
- Category.
- Estimated duration.
- Objective.
- Suggested topics.
- Suggested materials.
- Suggested exercise.

## Usage Flow

1. Mentor creates or edits a plan.
2. Mentor opens template picker.
3. Mentor selects one or more templates.
4. System copies templates into the plan as editable lessons.

## Acceptance Criteria

- Templates speed up plan creation.
- Copied lessons can be customized per mentee.
- Updating a template later does not rewrite completed lesson history.

## Global Design Direction

All UI-facing implementation from this spec must follow `plan/07-visual-identity-and-design/`. That folder is the official NossaMentoria visual identity: warm light surfaces, learning-oriented accents, Sora headings, Inter operational text, restrained motion, landing-first public entry, and dashboard-first private usability.
