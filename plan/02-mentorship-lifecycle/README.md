# Mentorship Lifecycle

This phase describes how a mentorship starts, becomes structured, runs through lessons, and tracks progress.

## Specs

- `picking-and-inviting-students.md`: how mentors connect with mentees.
- `mentorship-plan-spec.md`: how plans are created and maintained.
- `lesson-template-spec.md`: reusable starter lessons.
- `progress-tracking-spec.md`: completion and progress behavior.

## Lifecycle

1. Mentor picks or invites a mentee.
2. Relationship becomes active.
3. Mentor creates a mentorship plan.
4. Mentor adds lessons and materials.
5. Sessions are scheduled.
6. Sessions are completed and recorded.
7. Progress is updated.
8. Plan is completed, paused, or extended.

## Acceptance Criteria

- A relationship can exist before a plan.
- A plan can be built from scratch or from templates.
- Progress is based on lessons and session records, not only manual text.
